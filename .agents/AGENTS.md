# 法考通关助手 - 项目规则

## 语言
- 所有用户界面、文档、注释使用中文
- 代码变量名、函数名使用英文

## 技术栈
- 前端：HTML + Vanilla CSS + JavaScript (单页应用)
- 不使用框架，保持轻量
- 数据存储：LocalStorage（离线可用）
- 部署：静态文件，可直接用浏览器打开

## 设计原则
- 移动端优先，响应式设计
- 深色模式为主，护眼
- 法考学习场景优化：长时间阅读、快速答题
- 动画轻量，不干扰学习

## 编码规范
- 语义化 HTML
- CSS 变量统一管理主题
- JavaScript ES6+ 模块化
- 所有题目数据结构化（JSON）
