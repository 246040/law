# 法考通关助手 - 产品规格书

## 概述

**产品名称**：法考通关助手（LawPass）
**一句话描述**：为程序员/工程师量身打造的法考刷题工具，用判断树和工程思维攻克法考。
**目标版本**：v1.0 MVP

## 用户故事

### 作为法考考生，我需要：

| ID | 用户故事 | 优先级 | 验收标准 |
|----|---------|--------|---------|
| US-01 | 按科目/章节选择题目练习 | P0 | 可选择刑法>财产犯罪>盗窃vs诈骗 等层级 |
| US-02 | 做题时看到题干和选项，选择后立即判分 | P0 | 选中后高亮正误，显示正确答案 |
| US-03 | 看到每题的详细解析、法条依据、判断口诀 | P0 | 包含：解析、陷阱、规则、法条 |
| US-04 | 错题自动收入错题本 | P0 | 错题本可按科目筛选、重做 |
| US-05 | 查看做题统计（正确率、各科分布） | P1 | 图表展示总体和分科正确率 |

| US-07 | 限时模拟模式（计时、不可回看解析） | P2 | 倒计时、交卷、成绩单 |

## 页面结构

### 1. 首页/仪表盘
- 今日学习数据（做题数、正确率）
- 快速开始：继续上次 / 开始新练习
- 学习进度概览
- 每日目标达成情况

### 2. 题库页
- 科目树形导航（刑法 > 财产犯罪 > 盗窃vs诈骗）
- 每个章节显示题目数量和完成进度
- 支持顺序练习 / 随机练习

### 3. 做题页
- 题干展示
- 四个选项（支持单选/多选）
- 提交按钮
- 提交后显示：正误判断、正确答案、解析、陷阱、法条
- 上/下一题导航
- 当前进度 (3/10)

### 4. 错题本
- 按科目/时间筛选
- 每条显示：题干摘要、错误原因分类、做错次数
- 支持重做
- 支持标记"已掌握"移出错题本

### 5. 统计页
- 总做题量
- 总正确率
- 各科目正确率雷达图
- 近7天做题趋势
- 错误类型分布

### 6. 判断树页
- 财产犯罪判断树（交互式）
- 点击节点显示详细说明
- 关联真题示例

## 交互规格

### 做题流程
```
进入题库 → 选择章节 → 显示第1题
    → 选择选项 → 点击提交
    → 显示判分结果 + 解析
    → 点击下一题 → 循环
    → 全部做完 → 显示本轮成绩单
```

### 错题回炉流程
```
进入错题本 → 选择科目/全部
    → 开始重做
    → 做对 → 标记正确次数+1
    → 连续做对3次 → 建议移出错题本
```
