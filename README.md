## 🧠 科学记忆算法

采用 **SM-2 间隔重复算法**（Anki 同款），不是让你从头到尾盲目刷一遍：

```
第1次做对 → 1天后复习
第2次做对 → 3天后复习
第3次做对 → 7天后复习
做错了 → 立刻重新安排
```

**原理**：在你即将遗忘时安排复习，用最少的时间达到最好的记忆效果。

## ❓ 常见问题

<details>
<summary><strong>Q: 换手机后数据会丢失吗？</strong></summary>

数据存在浏览器本地。换设备前在「设置」页导出 JSON 文件，新设备导入即可恢复。
</details>

<details>
<summary><strong>Q: 真的能离线用吗？</strong></summary>

是的。首次打开加载完成后，添加到主屏幕，之后无网络也能正常刷题。
</details>

<details>
<summary><strong>Q: 题库会继续更新吗？</strong></summary>

会持续扩充。刷新页面即自动加载最新数据。
</details>

<details>
<summary><strong>Q: 和 XX 付费 APP 比怎么样？</strong></summary>

法考通适合**客观题刷题训练**。如果你需要视频课、名师讲解、主观题批改，那需要配合其他资源。法考通的优势是免费、离线、有记忆算法。
</details>

## 🛠 技术信息

<details>
<summary>展开查看技术栈和项目结构</summary>

### 技术栈

| 层次 | 技术 | 说明 |
|------|------|------|
| 前端 | Vanilla JS (ES Modules) | 零框架依赖 |
| 状态管理 | 自研 Store（发布/订阅） | localStorage 持久化 |
| 离线 | Service Worker + PWA | Cache First 策略 |
| 记忆算法 | SM-2 | 间隔重复 |
| 可视化 | Canvas 2D | 零依赖图表引擎 |
| 样式 | CSS Variables | 主题化设计系统 |
| 部署 | GitHub Pages | 免费 CDN |
| 测试 | Vitest | 60 个单元测试 |

### 项目结构

```
law/
├── index.html              # 入口
├── manifest.json           # PWA 配置
├── sw.js                   # Service Worker
├── css/                    # 样式
├── js/
│   ├── app.js              # 主入口
│   ├── state.js            # 状态管理
│   ├── utils/
│   │   ├── sm2.js          # SM-2 算法
│   │   ├── charts.js       # Canvas 图表
│   │   ├── keyboard.js     # 快捷键
│   │   ├── dom.js          # DOM 工具
│   │   └── helpers.js      # 通用工具
│   └── pages/              # 页面模块
├── data/
│   ├── subjects.json       # 学科配置
│   ├── questions/           # 题库（14 个 JSON 文件）
│   ├── flashcards.json     # 速记卡片
│   ├── knowledge.json      # 知识体系
│   └── laws.json           # 重点法条
└── tests/                  # 单元测试（60 个）
```

### 本地运行
